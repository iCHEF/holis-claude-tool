# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.1] - 2026-01-07

### Fixed
- Updated README with correct installation instructions for single plugin repository
  - Changed from marketplace-based to direct GitHub installation
  - Added project settings method for team auto-install

## [1.0.0] - 2026-01-07

### Added
- Initial release
- `solve-asana-issue` skill for comprehensive Asana issue resolution workflow
  - Reads and analyzes Asana issues
  - Investigates root cause
  - Creates fix PRs with proper testing
  - Follows iCHEF PR conventions (develop branch, Asana link in description)
